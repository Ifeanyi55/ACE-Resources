## **Launch GCP Associate Cloud Engineer Quiz Pro**

To launch the interactive quiz app in the browser, run the following commands:

`cd gcp-associate-cloud-engineer-quiz-pro`

`npm install && npm run dev`

Based on your score, the Gemini-powered quiz will tell you areas where you need to work on or areas where there are still some knowledge gaps.
